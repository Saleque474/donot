Now that we installed Nexus Root Toolkit, we will install the device drivers on the computer so that we can communicate with it.

1. From the main window of the toolkit, click on Full driver installation guide Automatic + manual.

2. Now if this is the first time you use this device on this computer and you have never installed any drivers for android devices on this computer then skip to step 3, otherwise go through step 1 and 2 to delete any previously installed drivers.

3. Now you can pick any solution, I picked 1, this will automatically prompt you with a driver installer, just go through it and your device should be installed.

4. Now go ahead and follow the instructions to enable USB debugging then click on Full Driver Test.


Now your device will reboot and if the driver was installed successfully you will get a message saying that.
